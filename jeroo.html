Jeroo Content

Unit 1

Unit Description

In Unit 1, we learned how to move the jeroo around the island by using hop(). We also learned the pick() and toss() commands to pick flowers and then toss them to clear nets. We also had to be careful to not run the jeroo into water or nets.

Concepts Learned

  • Using commands to control where the Jeroo goes like hop()
  • Preventing the Jeroo from dying
  • Using commands pick() and toss() to clear flowers and nets
  • Using the command plant() to plant flowers

Unit 2

Unit Description

In Unit 2, we learned how to create methods to solve problems. Using methods shortened the code and the time we spent programming becuase we didn't have to repeat commands so many times. Then, we called the methods to be used in the main method.

Concepts Learned

  • Creating methods to solve problems
  • Calling multiple methods in the main method

Unit 3

Unit Description

In Unit 3, we started to use loops and if statements in our methods. By putting ifs inside loops, we could shorten the amount of code we wrote by a lot. We put the loops and ifs inside the methods and the main method.

Concepts Learned

  • Using loops
  • Combining methods, loops, and ifs to solve problems

Unit 4

Unit Description

In Unit 4, we had to use many commands and have certain commands called when specific conditions were met. For example, in the Chaos Lab, we had to use many ifs for spcefic conditions. Then, we also used recursion to make a flower box. The jeroo turned right when encountering water.

Concepts Learned

  • Using recursion
